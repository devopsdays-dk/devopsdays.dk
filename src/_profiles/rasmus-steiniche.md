Rasmus Steiniche is the CEO who Codes @ Neurospace.
He has done everything from writing java applications. doing SaaS applications before it was cool, creating one of Denmark's biggest multi-cloud strategies, to creating big data and machine learning platforms.
Rasmus has been part of the Cloud Native and DevOps movement since 2014 and has moved on to create cloud native data and machine learning platforms.
Rasmus is a CNCF organizer at Cloud Native Aarhus / Cloud Native Nordics and a CNCF Ambassador (Denmark).
